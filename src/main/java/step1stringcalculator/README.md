## 기능 명세
* 문자열 입력 받기.v
* 결과값 출력
* 덧셈 v
* 뺄셈 v
* 곱셈 v
* 나눗셈 v
  * 결과 값 정수로 떨어지는 값으로 한정하기
    * 나누고서 버림이나 반올림등 처리 말하는 것으로 보임.
* 입력 문자열 숫자와 사칙연산 사이 빈 공백 문자열 존재하는 것으로 가정.

* 입력값에 따라서 계산순서 진행하기
  * 문자열 순서대로 사칙연산 진행하기
* 입력값 null이나 공백이면 IllegalArgumentException v
* 문자열이 아닌 값이 있으면 Illegal v
* 기호가 없으면 Illegal v
