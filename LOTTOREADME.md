## 기능 명세
*[ ] 구입금액 입력 받기.
  *[ ] 금액을 개수로 변환
*[ ] 회차별 로또 랜덤 번호 6개 생성
  * [ ] shuffle 사용해서 랜덤 번호 생성
  * [ ] sort 사용해서 마지막에 데이터 정렬
*[ ] 당첨번호 입력받기
* [ ] 각회차마다 당첨번호와 몇개나 일치하는지 판단
  * [x] contains사용해서 일치 횟수 찾기
* [x] 일치하는 횟수에 따라 당첨별 횟수 저장하기
* [ ] 당첨금액/구입금액 계산하기( 버림 사용)
